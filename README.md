Project: Small Business Network Design & Implementation

Designed and implemented a secure branch office network for a small business using Cisco router and switch

Created multiple VLANs for Admin/IT, Finance/HR, and Customer Service departments to ensure network segmentation

Configured inter-VLAN communication, DHCP for automatic IPv4 addressing, and wireless access for each department

Used 192.168.1.0/24 addressing scheme and ensured full connectivity across departments

Tools & Technologies: Cisco Packet Tracer, VLANs, Routing & Switching, DHCP, Wireless Networking, TCP/IP
